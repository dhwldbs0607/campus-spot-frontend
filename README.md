Campus Spot Core
위치 기반 실시간 커뮤니티 플랫폼 (핵심 로직 프로토타입)

[프로젝트 개요]
특정 장소(Spot)에 접근 시 활성화되는 위치 기반 게시판과 도움 요청(SOS) 리워드 시스템을 구현한 웹 서비스입니다.
복잡한 외부 라이브러리를 배제하고, 핵심적인 데이터 흐름(Data Flow)을 직관적으로 보여주기 위해 순수 React로 재구성했습니다.

[핵심 기능 및 비즈니스 로직]

실시간 뷰 스위칭

구현 내용: GPS 위치(Spot) 변경 시 하단 게시판 데이터 실시간 동기화

비즈니스 의의: 위치 데이터와 UI 화면 간의 실시간 무결성 증명

리워드 즉시 연동

구현 내용: 타인의 SOS 요청 수락 시, 사용자 계정에 포인트 즉시 가산

비즈니스 의의: 부채(비용) 증가로 이어지는 보상 시스템의 트리거 및 누적 과정 시각화

참고: 해당 리워드 시스템의 어뷰징 방어 및 내부통제 데이터 분석 로직은 별도 저장소에 분리하여 관리 중입니다.

[기술 스택]

Frontend: React, TypeScript

Styling: Inline CSS (즉각적인 구동 테스트 목적)

Architecture: App.tsx 단일 파일 내에서 Input 부터 Logic, State, Output 으로 이어지는 데이터 흐름 구현
