# 📍 Campus Spot Core (UI/UX & 핵심 로직 프로토타입)

## 1. 프로젝트 개요
'캠퍼스 스팟(Campus Spot)'은 특정 교내 장소에 접근했을 때만 활성화되는 **위치 기반 실시간 커뮤니티 플랫폼**입니다. 
본 저장소는 전체 웹 시스템 중 '위치 기반 게시판 활성화 로직'과 '사용자 간 SOS 상호작용 및 리워드 지급 시스템'의 데이터 흐름을 직관적으로 보여주기 위해 불필요한 종속성을 제거하고 순수 React로 재구성한 핵심(Core) 프로토타입입니다.

## 2. 비즈니스 로직 및 핵심 기능

### 🎯 [1] GPS 위치 시뮬레이션 및 동적 렌더링
- 사용자의 현재 위치(Spot) 데이터가 변경됨에 따라, 하단 게시판의 데이터가 해당 장소의 실시간 메시지로 즉각 스위칭되는 로직을 구현했습니다.
- **의의:** 위치 데이터와 뷰(View)의 완벽한 동기화를 시각적으로 증명합니다.

### 💰 [2] SOS 상호작용 및 리워드(보상) 시스템 연동
- 사용자가 타인의 '도움 요청(SOS)'을 수락하여 완료 상태로 변경할 경우, 즉각적으로 사용자 계정에 리워드 포인트가 가산되는 로직을 구현했습니다.
- **의의:** 기업의 재무적 비용(부채)으로 직결되는 리워드 시스템이 프론트엔드 환경에서 사용자 액션에 따라 어떻게 트리거(Trigger)되고 누적되는지 그 데이터 흐름을 명확히 보여줍니다. (※ 해당 리워드 시스템의 어뷰징 방어 및 내부통제 로직은 별도의 데이터 분석 저장소에서 다루고 있습니다.)

## 3. 기술 스택 (Tech Stack)
- **Frontend:** React, TypeScript
- **Styling:** Inline CSS (외부 라이브러리 의존성 없이 즉각적인 구동 및 테스트가 가능하도록 설계)
- **Architecture:** `App.tsx` 단일 파일 내에서 Input ➡️ Logic ➡️ State ➡️ Output으로 이어지는 데이터 흐름을 투명하게 구현.
